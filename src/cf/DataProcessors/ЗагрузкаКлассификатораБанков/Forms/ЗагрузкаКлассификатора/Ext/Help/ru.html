Программа позволяет загружать классификатор банков РФ с сайта 1С. При этом программа учитывает недействующие банки (в карточке такого банка включен соответствующий флажок).

Загрузка классификатора банков РФ

Производится с помощью кнопки Загрузить классификатор из классификатора банков.

  • С помощью переключателя выберите Вариант загрузки:

    • С сайта 1С;

    • с диска "Информационно-технологическое сопровождение" - загрузка будет производиться из поставляемого на диске 1С:ИТС "Общероссийского классификатора банков" (ОКБ).

      • укажите Путь к диску ИТС;

    • Из файла.

      • укажите путь к файлу на диске, например, E:\Классификатор банков\bnk.zip. Скачать файл с классификатором банков можно по адресу https:\\bankregister.1c.ru\bankregister/v1/bnk.zip (для доступа потребуются логин и пароль от ИТС).

  • Нажмите Загрузить.

  • После успешной загрузки программа выводит сообщение о количестве загруженных банков. Нажмите Закрыть.

Повторная загрузка

  • При повторной загрузке перезаписываются только измененные сведения классификатора.
  • При этом если был изменен БИК банка, то будет создана новая запись о банке.
  • Если были изменены другие поля, то при повторной загрузке они будут перезаписаны (кроме полей Адрес и Телефоны в случае, когда классификатор перезагружается с сайта 1С).

Особенности загрузки классификатора банков в веб-клиенте

  • В веб-клиенте также производится проверка актуальности классификатора банков и выводится сообщение с предложением его обновления.
  • Если в программе до публикации в веб-клиенте был загружен классификатор банков, то при работе в веб-клиенте необходимо будет его только обновить.
  • Загрузка с диска в веб-клиенте недоступна.
